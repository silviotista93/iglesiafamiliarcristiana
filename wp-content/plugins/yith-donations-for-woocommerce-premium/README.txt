YITH Donations for WooCommerce allows you to add donation in your orders

== Description ==

Add to your shop the possibility for your users to make donations for a good cause or an important solidarity campaigns. Make your users in the position to fill a donation form, make a donation in your site and let them set the amount they want to donate. Now you can do that with YITH Donations for WooCommerce.

**Main features:**

* Give a name to each donation project
* Associate a donation form to a single product
* Donation widget to add to your sidebars

= Installation =

Once you have installed the plugin, you just need to activate it to enable it.

= Configuration =

YITH Donations for WooCommerce will add a new submenu called "Donations" in "YIT Plugins" menu. From there, you can configure all plugin settings.
You will be able to add your donation in your sidebar thanks to a useful widget.

== Installation ==

1. Unzip the downloaded zip file.
2. Upload the plugin folder into the `wp-content/plugins/` directory of your WordPress site.
3. Activate `YITH Donations for WooCommerce` from Plugins page
